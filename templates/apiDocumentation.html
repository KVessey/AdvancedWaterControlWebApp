API Documentation

Device API

{URL}:{PORT}/devicesApi
This request retrieves the list of all devices in the Database

REQUEST - GET: localhost:8090/devicesApi

RESPONSE -

{
"devices": [
{
"id": 1,
"ip_address": "192.168.0.1",
"last_watered": "01:00",
"name": "Kyle Pi",
"restart_time": "01:00"
},
{
"id": 2,
"ip_address": "172.168.0.1",
"last_watered": "10:10",
"name": "API Demo Device",
"restart_time": "10:10AM"
} ]
}
This request updates the device based on ID

REQUEST - PUT: localhost:8090/devicesApi/1

RESPONSE -

{
"devices": [
{
"id": 1,
"ip_address": "192.168.0.1",
"last_watered": "01:00",
"name": "Kyle Pi",
"restart_time": "01:00"
} ]
}
This request updates the device based on ID

REQUEST - POST: localhost:8090/devicesApi/1

RESPONSE -

{
"devices": [
{
"id": 1,
"ip_address": "192.168.0.1",
"last_watered": "01:00",
"name": "Kyle Pi",
"restart_time": "01:00"
} ]
}
This request deletes the device based on ID

REQUEST - DELETE: localhost:8090/devicesApi/1

RESPONSE -

Removed device with ID 1
